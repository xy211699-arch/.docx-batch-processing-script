# .docx-batch-processing-script
一个Python脚本，用于批量自动化处理Word文档，实现文档格式化、智能图片插入等功能
## 创建说明
项目起因是之前接收了一个实习项目，具体内容是搜集200位中外名人因病去世的信息，整理后按规定格式放入word文档，并且一位名人对应一个word文档，考虑到工作量巨大，于是借助ai工具生成了这样一份批处理脚本
## 项目说明
该脚本具有文档格式化，页码添加，智能图片插入，文件管理等功能，并具有用户便利的运行指导
## 输入与输出
### 输入
文件格式：.docx（仅支持）
命名格式：任意
### 输出
处理后的文档：对原文档进行替换
备份文件：自动创建备份文件夹（可选）
人物图片：保存在桌面“人物图片”文件夹
错误报告：如有失败，生成详细报告
# 错误说明
图片有误：该项目的智能图片插入功能主要从必应图片中搜索人物对应的图片，匹配筛选功能不尽完善，可能存在实际图片与对应人物不符的情况
插入有误：部分检索的图片格式（例如.wbpg）可能无法导入word中
人物检索有误：请确保文档第一行格式为：中文姓名（英文姓名）
